# Factorial Generator

A simple Python program that calculates the factorial of a number entered by the user.  
The program shows the calculation process step by step with a short delay for better visualization.

---

## Description
This program takes a non-negative integer as input and returns its factorial.  
It demonstrates the use of loops, functions, and basic time delay in Python.

---

## Example Output
``` bash
Factorial Generator
Enter a positive integer: 5

Processing...

Calculating: 1! = 1
Calculating: 2! = 2
Calculating: 3! = 6
Calculating: 4! = 24
Calculating: 5! = 120

The factorial of 5 is: 120
```

---

## How to Run
1. Clone or download this repository.
2. Open your terminal in the project directory.
3. Run the program with:
``` bash
python factorial.py
```
